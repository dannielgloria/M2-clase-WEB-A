Selectores básicos

Los selectores básicos en CSS te permiten seleccionar elementos HTML para aplicar estilos. Algunos de los selectores más comunes incluyen:

  • Selector de tipo (etiqueta): selecciona todos los elementos de un tipo específico, por ejemplo, p para seleccionar todos los párrafos.
  • Selector de clase: selecciona elementos con una clase específica, por ejemplo, .mi-clase para seleccionar todos los elementos con la clase "mi-clase".
  • Selector de ID: selecciona un elemento con un ID específico, por ejemplo, #mi-id para seleccionar el elemento con el ID "mi-id".

Selectores universales y combinaciones

Los selectores universales y combinaciones te permiten seleccionar elementos de manera más específica. Algunosejemplosincluyen:

  • Selector universal: selecciona todos los elementos, por ejemplo, *.
  • Combinaciones de selectores: puedes combinar selectores para ser más específicos, por ejemplo, div p para seleccionar todos los párrafos dentro de un div.

Display y tipos de cajas

El modelo de caja en CSS describe cómo se representan los elementos en la página. El tipo de display determina cómo se comporta un elemento. Algunos tipos comunes de display incluyen:

  • Block: el elemento ocupa todo el ancho disponible y comienza en una nueva línea, por ejemplo, div.
  • Inline: el elemento solo ocupa el espacio necesario y no comienza en una nueva línea, por ejemplo, span.
  • Inline-block: combina características de block e inline, permitiendo que el elemento tenga un tamaño específico pero sin forzar un salto de línea.

Pseudo clases

Las pseudo clases en CSS te permiten aplicar estilos a elementos en estados específicos o con ciertas características. Algunos ejemplos comunes de pseudo clases incluyen:

  • :hover : se aplica cuando el usuario pasa el cursor sobre un elemento.
  • :focus : se aplica cuando un elemento tiene el foco (por ejemplo, un campo de entrada).
  • :active : se aplica al elemento activo (por ejemplo, un botón presionado).
